# IGDTUW Review Bot
> A smart and interactive bot to fetch, summarize, and review information related to IGDTUW projects, cultural events, and student resources.

IGDTUW Review Bot is a personalized assistant bot designed to help students and developers explore, summarize, and access IGDTUW-related information quickly. It focuses on providing project insights, cultural event summaries, and branch-specific guidance for easy reference.

Features include fetching reviews and summaries from IGDTUW repositories and resources, branch-specific project details and recommendations, cultural events and college fest insights, and an easy-to-use interface for interactive queries. The bot can also be extended with AI-powered suggestions.

To set up, clone the repository using `git clone https://github.com/Ishikaaa2502/igdtuw-review-bot.git`, navigate to the project folder with `cd igdtuw-review-bot`, install dependencies with `npm install`, and configure any required API keys (like Google Generative AI).

Run the bot locally using `node index.js` and follow the prompts to interact. Use commands to fetch project summaries, cultural insights, or branch-specific guides.

Technologies used: Node.js for runtime, JavaScript for core logic, Google Generative AI API for content generation, and VSCode for development.
